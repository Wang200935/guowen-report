# Presentation Style Guide Bilingual Summary

## Chinese

### 這份簡報風格的本質

這個倉庫最值得學的，不是原本的主題，而是簡報製作方式。

核心特徵是：

- 觀點先行
- 結構清楚
- 每頁只講一件事
- 標題像結論
- 圖片輔助內容，不主導內容
- 整份簡報有穩定節奏

### 最穩定的頁型

最常用、也最穩定的頁型有：

1. 封面頁
2. 總覽頁
3. 主體分析頁
4. 統整頁
5. 結論頁

其中主體分析頁最適合用：

- 左側文字
- 右側主圖
- 右側摘要卡

### 標題怎麼寫

不要把標題寫成章節名稱。

要寫成：

- 一句判斷
- 一個觀察
- 一個觀點

這樣整份簡報才會看起來像真的有思考過，而不是只是整理資料。

### 圖片怎麼用

圖片應該：

- 幫助理解
- 補充氣氛
- 提供證據感

圖片不應該：

- 只是漂亮
- 搶走文字重點
- 讓畫面變亂

### 視覺感覺

這個倉庫最後做出來的視覺偏向：

- 深色背景
- 淺色文字
- 少量亮色強調
- 卡片式資訊整理
- 科技簡報感

### 一句話總結

> 先把觀點想清楚，再用固定頁型、大標題、少量重點與單一主圖，把內容做成有節奏、低噪音、可快速理解的簡報。

## English

### The essence of this presentation style

The most reusable part of this repository is not its original topic, but its presentation method.

The core traits are:

- opinion-first
- clear structure
- one main idea per slide
- titles that read like conclusions
- images that support the argument instead of driving it
- a stable slide-to-slide rhythm

### The most stable slide types

The most useful recurring slide types are:

1. cover slide
2. overview slide
3. body analysis slide
4. synthesis slide
5. conclusion slide

The body slide works best with:

- text on the left
- one main image on the right
- small summary cards on the side

### How titles should be written

Do not write titles like section labels.

Write them as:

- a judgment
- an observation
- a point of view

That is what makes the deck feel intentional rather than merely organized.

### How images should be used

Images should:

- improve understanding
- reinforce mood
- add evidence value

Images should not:

- exist just because they are pretty
- overpower the main point
- make the slide noisy

### Visual feel

The final visual direction in this repository leans toward:

- dark backgrounds
- light text
- limited accent colors
- card-based information grouping
- a tech-presentation feel

### One-sentence summary

> Clarify the argument first, then use repeatable slide types, large titles, a small number of key points, and one dominant image to create a rhythmic, low-noise, fast-to-understand presentation.
